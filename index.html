¿Qué aprenderás?

Este mini curso gratuito da una visión general de Spark y el módulo Spark SQL para manejo de datos estructurados.

  • Introducción a Spark
  • Instalación de herramientas
  • Conexión a bases de datos y lectura de archivos
  • Consultas básicas de datos
  • Filtrar datos
  • Agrupaciones de datos

¿Para quién es este curso?

Este curso esta diseñado para personas interesadas en aprender sobre manejo de datos estructurados usando el framework de Spark en Python. Es deseable tener conocimientos básicos de programación en Python y de SQL. El curso se dictará vía zoom y las grabaciones quedarán disponibles después

Tutora

Este curso será dictado por Paula Lago, PhD

Paula un doctorado en Ingeniería y sus intereses de investigación incluyen el análisis de datos de sensores, la computación ubicua y su aplicación en el sector de la salud.
