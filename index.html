¿Para quién es este curso?

Este curso esta diseñado para personas interesadas en aprender sobre manejo de datos estructurados usando el framework de Spark en Python. Es deseable tener conocimientos básicos de programación en Python y de SQL. El curso se dictará vía zoom y las grabaciones quedarán disponibles después
